# Hackathon — Learn to code with Python!

## How to start
1. Open Pickcode (https://pickcode.io) and sign up
2. Start with 00_basics — just read and run each file
3. Move to 01_turtle when you're ready to draw
4. Pick any game in 04_pygame_games and start hacking!

## Rules
- There are no rules
- Change anything you want
- Break things — that's how you learn
- Ask for help whenever you're stuck
