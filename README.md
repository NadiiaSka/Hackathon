## How to start

1. Open Pickcode (https://pickcode.io) and sign up.
2. Start with [00_basics](00_basics) - read and run each file.
3. Move to [01_turtle](01_turtle) when you are ready to draw.
4. Pick any game in [04_pygame_games](04_pygame_games) and start hacking.
